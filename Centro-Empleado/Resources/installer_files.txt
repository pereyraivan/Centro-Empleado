---------------------

CARPETA DE DESTINO RECOMENDADA:
C:\Program Files\Centro-Empleado\

PERMISOS REQUERIDOS:
- Instalación: Administrador
- Ejecución: Usuario normal
- Base de datos: Se crea automáticamente

NOTAS IMPORTANTES:
- La base de datos se crea automáticamente al primer uso
- No se requiere configuración manual
- Todos los archivos SQLite deben estar en la misma carpeta
- La aplicación funciona inmediatamente después de la instalación

VERIFICACIÓN POST-INSTALACIÓN:
1. Ejecutar setup_database.bat
2. Verificar que todos los archivos estén presentes
3. Ejecutar Centro-Empleado.exe como administrador la primera vez
4. Confirmar que se crea CentroEmpleado.db automáticamente
